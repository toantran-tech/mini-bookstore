# 📚 Mini Bookstore
> Full-stack bookstore app | Spring Boot 3 + JWT + React 19

## 🔗 Live Demo
- Frontend: https://mini-bookstore.vercel.app
- API Docs: https://xxx.railway.app/swagger-ui.html

## ✨ Features
- JWT Authentication (Register/Login)
- Role-based access (Admin/User)
- Book catalog with search & pagination
- Shopping cart & order management
- Admin panel (CRUD books, manage orders)

## 🛠 Tech Stack
| Layer | Tech |
|-------|------|
| Backend | Spring Boot 3, Spring Security 6, JWT |
| Database | MySQL 8 |
| Frontend | React 19, Vite, Tailwind CSS v4 |
